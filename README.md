# Hospital Management Prototype Submission for Smart India Hackathon 2024 Internal Hackathon Round 1
This project aims to build a Healthcare Management System inspired by the ZK Medical Billing Platform. This offers comprehensive features such as Disease Detection, providing recommendations while always having a multilingual chatbot.
